    Class FaultHandlingModule

    Registers interception such that types annotated with RetryOnFaultAttribute will have their virtual members intercepted by RetryInterceptor, which will retry failed operations with an exponential back-off calculated by ExponentialRetryBackoffCalculator.

    Assumes that zero or more instances of IExceptionRetryPredicate are registered, which help shape the retry behavior.

    Assumes an is resolvable, with a section named FaultHandling that is mappable to FaultHandlingConfiguration.
