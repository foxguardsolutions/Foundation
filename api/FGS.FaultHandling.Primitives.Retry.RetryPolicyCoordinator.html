    Class RetryPolicyCoordinator

    An implementation of IRetryPolicyCoordinator that satisfies requests for IRetryPolicy by checking first-time exceptions against IExceptionRetryPredicate instances, in order to determine whether or not a retrying retry policy is returned to the caller.
