    ProceedAsync()

    Asynchronously proceeds the call to the next interceptor in line, and ultimately to the target method.

    Declaration
    Task ProceedAsync()
    Returns
    Type Description
    Task

    A Task to await the completion of the method invocation.

    The returned task will not contain the return value of the invocation. For that, use the ReturnValue property.
    Remarks

    To get or set the invocation return value, use the ReturnValue property.

    Since interface proxies without a target don't have the target implementation to proceed to, it is important, that the last interceptor does not call this method, otherwise a will be thrown.
